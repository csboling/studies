This code uses several libraries. It was developed on Fedora 19.
The following dependencies should be available as packages on many 
modern Linux distributions:
  numpy        -- for general-purpose vector management
  scipy        -- for signal processing
  scikit-learn -- for machine learning/optimization solvers
  matplotlib   -- for plotting
The following Python modules are listed in the Python third-party module
repositories and should be available using easy_install:
  PyWavelets
  bitstring
